# HTML

html 이란?

* 표준 웹 페이지 마크업 언어

* 웹 브라우저가 어떻게 보여주어야하는지 나타내는 언어

* element의 구성을 표현한 언어

### Element

element는 브라우저에서 컨텐츠를 어덯게 표시하는지를 알려준다.

element의 형식은 다음과 같다.

```html
<tagname> Content goes here... </tagname>
```

### HTML Structure

```html
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph./p>
</body>
</html>
```
